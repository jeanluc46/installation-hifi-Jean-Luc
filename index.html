Source et Contrôle

Routeur Bbox WiFi 7

Point d'entrée du réseau domestique.

iPad Pro M2 11"

Interface de contrôle sans fil (WiFi 7) permettant de piloter l'ensemble du système via les applications :

  • GentooPlayer
  • HQPlayer
Routeur
iPad Pro
via Ethernet avec filtre réseau
2
Filtrage Réseau

Filtre Network Acoustics Eno 2

Positionné entre le routeur et le serveur.

Connexion : Câble réseau BIVANI CAT 8 audiophile, optimisé pour réduire les interférences.

Filtre
via Câble Réseau Audiophile
3
Cœur du Système : Serveur Audio

Serveur HDPlex H5 Fanless

Ordinateur audiophile passif (sans ventilateur) dédié à la lecture musicale haute résolution.

Configuration interne :

  • SSD NVMe 1 To : Système d'exploitation GentooPlayer optimisé
  • SSD NVMe 4 To : Bibliothèque musicale
  • Carte JCAT NET Card XE EVO : carte réseau ethernet Audiophile synchronisée avec masterclock 20 mhz
  • Carte JCAT USB XE EVO : carte USB Audiophile synchronisée avec masterclock 20 mhz
  • Câblage interne Mundorf/Duelund : Fils de haute qualité pour les connexions internes

Sortie : USB Type B vers le convertisseur µDDC via câble Tubulus Concentrus

Serveur
via USB Audiophile
4
Synchronisation Masterclock

Masterclock By Watt Cat

Horloge de référence OCXO (Oven-Controlled Crystal Oscillator) ultra-stable.

Caractéristiques :

  • Horloge OCXO ultra-stable
  • 3 sorties BNC 50 ou 75 ohms
  • Sélection sinus ou carré sur chaque sortie BNC
  • Génération de multiples fréquences d'horloge

Synchronisation :

  1. Carte JCAT NET Card XE EVO → 20MHz (BNC 50Ω)
  2. Carte JCAT USB XE EVO → 20MHz (BNC 50Ω)
  3. LAiV Harmony µDDC → 10MHz (BNC 50Ω)

Câbles BNC Harmonic Technology 50 Ohms sur les 3 sorties

Masterclock
5
Conversion USB → I2S

LAiV Harmony µDDC

Convertisseur USB vers I2S de très haute qualité, transformant le signal USB du serveur en signal I2S natif pour le DAC.

Connexions :

  • Entrée : USB Type B depuis le serveur (câble Tubulus Concentrus)
  • Sortie : I2S HDMI vers le DAC (câble Tubulus Concentrus)
µDDC
6
Conversion Numérique-Analogique

DAC Holo Spring 3 Level 2

Convertisseur numérique-analogique de référence avec architecture R2R discrète.

Caractéristiques :

  • Architecture R2R discrète
  • Entrée I2S HDMI privilégiée

Connexions :

  • Entrée : I2S HDMI depuis le µDDC via câble Tubulus Concentrus I2S
  • Sortie : Signal analogique RCA vers l'amplificateur
DAC
via Signal Audio Analogique

Amplification

Amplificateur Sugden A21 Signature (Class A)

Amplificateur intégré haute-fidélité en classe A pure.

Connexions :

  • Entrée : RCA depuis le DAC via câble de modulation Tubulus Concentrus RCA
  • Sortie : Enceintes via câble HP JSV Audio Signature 2
Ampli
via Câbles Haut-Parleurs

Enceintes Colonnes JMR Euterpe Suprême
